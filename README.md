# GtKram

A web app for organising children's bazaars

## Frontend

* Razor Pages in ASP.NET Core
* Bulma Framework
* jQuery 
* Font Awesome
* QR Code reading
 
## Backend

* .NET Core 10.0
  * ASP.NET Core
* Template Engine with Scriban
* Logging via Serilog
* SQLite Persistence with Dapper ORM
* Madiator Pattern
