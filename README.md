# GT Kram

A web app for organising children's bazaars

## Frontend

* Bulma Framework
* jQuery 
* Font Awesome
* QR Code generation
 
## Backend

* .NET Core 10.0
* ASP.NET Core Razor Pages
* Template Engine with Scriban
* Logging via Serilog
* SQLite Persistence with Dapper ORM
* Madiator Pattern
